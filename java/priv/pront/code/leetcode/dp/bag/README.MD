# 背包问题总结

### 求最大价值
> dp[i][j] = Math.max(dp[i - 1][j] + dp[i - 1][j - w[i]] + v[i])
### 求最小价值
> dp[i][j] = Math.min(dp[i - 1][j] + dp[i - 1][j - w[i]] + v[i])


### 求符合条件的方法数、求个数
> **01背包举例** ：dp[i][j] = dp[i - 1][j] + dp[i - 1][j - w[i]]
* 求个数后面就不加价值了。

### 返回boolean
> 实际上就是dp[n - 1][m - 1]和题干上面判断
